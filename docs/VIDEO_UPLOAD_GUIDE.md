# رفع الفيديوهات (رفع مباشر) — Bunny Stream (الموصى به)

بسبب أن Cloudinary قد لا يكون متاحاً في بعض الدول، ننصح باستخدام **Bunny.net Stream** لبث فيديوهات الكورسات.

---

## 1) إنشاء حساب Bunny
- افتح bunny.net
- Create account

## 2) إنشاء Video Library
- Dashboard → Stream
- Create Video Library
- اختر Region قريبة (Europe غالباً ممتاز)
- فعّل Security (Token auth) إن رغبت

## 3) رفع الفيديو (كل الصيغ)
- داخل Video Library → Upload
- ارفع أي صيغة: MP4 / MOV / AVI …

## 4) أخذ رابط التشغيل
بعد الرفع، Bunny يعطيك رابط تشغيل (Playback URL) أو Embed.
انسخ رابط التشغيل (URL) واستخدمه في Sanity.

## 5) ربط الفيديو داخل الكورس
- افتح Sanity Studio
- Lessons / Episodes
- افتح الحلقة
- ضع الرابط داخل: **Video URL**
- Publish

> ملاحظة: الأفضل استخدام رابط تشغيل رسمي من Bunny (Playback URL) لضمان سرعة واستقرار أعلى.
